### 기대효과
아이템이 효력이 나타나게 하기 위해서는 자바스크립트 문제를 풀어야 하기 때문에 자연스럽게 자바스크립트에 대한 공부를 할 수 있으며 문제를 풀면 획득한 아이템의 능력을 사용할 수 있기 때문에 성취감을 얻을 수 있다. 게임도 하면서 자바스크립트 공부도 할 수 있기 때문에 일석이조의 효과가 있다.
